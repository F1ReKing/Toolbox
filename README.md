<p>Android 开发者工具类集合</p>

## 引入

#### Step 1. Add the JitPack repository to your build file
```
	allprojects {
		repositories {
			...
			maven { url "https://jitpack.io" }
		}
	}
```

#### Step 2. Add the dependency
```
	dependencies {
	       implementation 'com.github.F1ReKing:Toolbox:1.0.0'
	}
```
